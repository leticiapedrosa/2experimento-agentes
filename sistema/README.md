## Sistema (Exam Question Management)

This repository is **Phase 1 (scaffolding & requirements)** for a university experiment.

- **client/**: React + TypeScript + Tailwind (plumbing only)
- **server/**: Node + TypeScript + Cucumber (Gherkin) acceptance tests (pending steps)

### Commands

Install dependencies (from `sistema/`):

```bash
npm install
```

Run server acceptance tests (they will be pending until implementation exists):

```bash
npm run test:acceptance
```

Run client dev server:

```bash
npm run dev:client
```

Run server dev entry (placeholder):

```bash
npm run dev:server
```
